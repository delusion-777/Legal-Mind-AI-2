---

## 📌 Overview

**LGLmind** is an AI-powered Legal Document Assistant developed using **Python, HTML, CSS, JavaScript, and Hugging Face models**.

The platform enables users to interact with legal documents through intelligent question answering, generate concise summaries, analyze legal risks, and transform legal text for easier understanding.

LGLmind simplifies complex legal language and improves document accessibility through AI-powered automation.

---

## 🚀 Features

### 🧠 AI Chatbot (Question Answer Retrieval)
- Ask questions from uploaded legal documents
- Retrieval-based intelligent responses
- Context-aware legal document understanding
- Fast and efficient document interaction

### 📄 Legal Document Summarizer
- Generate concise summaries of lengthy legal documents
- Extract key legal information instantly
- Simplifies legal content for better readability

### 🔍 Risk Analyzer
- Detects potential legal risks in documents
- Identifies important legal concerns
- Helps users understand possible risks in agreements

### 🎧 EchoVerse
- AI-powered legal text transformation
- Rephrases legal text intelligently
- Improves readability of complex legal content

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python
- **AI/ML Models:** Hugging Face Transformers
- **Natural Language Processing (NLP)**
- **Question Answer Retrieval System**

---

## 🤖 AI Integration

LGLmind uses **Hugging Face models** for:

- **Chatbot (Q&A Retrieval)** → Intelligent document-based question answering
- **Summarizer** → Legal document summarization
- **EchoVerse** → Text transformation and enhancement

These models help process legal language efficiently and generate meaningful responses.

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/LGLmind.git
cd LGLmind
```

### 2️⃣ Create Virtual Environment (Optional)

```bash
python -m venv venv
```

Activate environment:

#### Windows
```bash
venv\Scripts\activate
```

#### Mac/Linux
```bash
source venv/bin/activate
```

---

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

Run the application:

```bash
python app.py
```

Then open:

```bash
http://127.0.0.1:5000/
```

in your browser.
